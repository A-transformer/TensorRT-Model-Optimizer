========================
Installation for Windows
========================

**System Requirements**

The following system requirements are necessary to install and use TensorRT Model Optimizer - Windows:

+-------------------------+-----------------------------+
| OS                      |  Windows                    |
+-------------------------+-----------------------------+
| Architecture            |  amd64 (x86_64)             |
+-------------------------+-----------------------------+
| Python                  |  >=3.10,<3.13               |
+-------------------------+-----------------------------+
| CUDA                    |  >=12.0                     |
+-------------------------+-----------------------------+
| ONNX Runtime            |  1.20.0                     |
+-------------------------+-----------------------------+
| Nvidia Driver           |  565.90 or newer            |
+-------------------------+-----------------------------+
